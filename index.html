As Principais Linguagens como Desenvolvedor de Sistemas (1semestre)

Html e CSS

HTML é a linguagem de marcação padrão para criar páginas web, Já o CSS é uma linguagem de estilo usada para controlar a apresentação e o layout de documentos HTML.

Java Script

Ele sendo principalmente conhecido por ser a linguagem de script dos navegadores web, permitindo uma melhor interatividade com páginas web

C++

C++ é uma linguagem de programação de propósito geral que foi desenvolvida como uma extensão da linguagem de programação C, conhecida por ser uma linguagem de programação de alto desempenho, orientada a objetos.

JSON

(JavaScript Object Notation) é uma programação para representar dados de maneira estruturada. Embora tenha "JavaScript" em seu nome, JSON é independente de linguagem e é amplamente usado em diversas linguagens de programação.
